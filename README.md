# 🖥️ Projeto de Autoatendimento — Serviços Públicos

> Interface web de autoatendimento para facilitar o acesso da população a serviços públicos de forma autônoma, ágil e intuitiva.

---

## 📋 Visão Geral

O **Projeto de Autoatendimento** é uma solução digital voltada para a modernização do atendimento em órgãos e instituições públicas. Por meio de uma interface web acessível via navegador, o cidadão pode realizar solicitações, consultas e interações com serviços públicos sem a necessidade de atendimento presencial ou intermediação humana.

O projeto foi desenvolvido com foco em **acessibilidade**, **usabilidade** e **eficiência**, garantindo uma experiência fluida tanto para usuários com familiaridade digital quanto para aqueles com menor letramento tecnológico.
---

## ✨ Funcionalidades Principais

* **Dashboard de Acompanhamento:** Visualização em tempo real do status das solicitações:
    * 🟡 **Pendentes:** Aguardando análise inicial.
    * 🔵 **Em andamento:** Processos sendo analisados ou executados.
    * 🟢 **Resolvidas:** Solicitações concluídas e finalizadas.
* **Nova Solicitação:** Fluxo rápido para abertura de novos chamados.
* **Central de Consultas:** Botão "Ver todas" para acesso ao histórico completo de interações.
* **Interface Responsiva:** Otimizada para Desktop e dispositivos móveis.
* **Acessibilidade (WCAG 2.1):** Contraste adequado, navegação por teclado e linguagem simplificada.

---

## 🎯 Objetivo

Reduzir filas, tempos de espera e burocracia no acesso a serviços públicos, oferecendo ao cidadão uma plataforma de autoatendimento disponível 24/7 diretamente pelo navegador.

---

## ✨ Funcionalidades

- ✅ Consulta de serviços disponíveis
- ✅ Solicitação e acompanhamento de processos
- ✅ Autenticação do cidadão
- ✅ Preenchimento de formulários digitais
- ✅ Geração e download de documentos / comprovantes
- ✅ Notificações de status e andamento
- ✅ Interface responsiva (desktop e mobile)
